# IntelliScan Use Case Diagrams for Draw.io

Below are 5 use case diagrams based on user roles and workflows from the IntelliScan project. Each is formatted as a prompt you can copy-paste into draw.io's AI diagram generator to create the visual diagram. No Mermaid code or colors are used. These prompts are refined for better compatibility with draw.io's AI.

## 1. Use Case Diagram for Anonymous Visitor Workflow
Generate a UML use case diagram showing an anonymous visitor interacting with public features. Include a stick figure actor labeled "Anonymous Visitor" on the left. Add oval use cases on the right: "View Public Pages", "Initiate Authentication", "Access Calendar Links", "Perform Event Kiosk Scan". Draw solid lines connecting the actor to each use case.
