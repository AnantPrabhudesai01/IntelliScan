IntelliScan - Activity Diagrams

Business Card Intelligence Platform - Workflow Analysis

1. Card Scanning & Contact Creation Workflow
Key Points: This workflow shows the complete card scanning process from user initiation through final confirmation. The three input methods (camera, upload, manual) converge into a unified processing pipeline. OCR extracts raw text, AI validates and structures the data with confidence scoring. If confidence is above 80%, the contact saves automatically. Below 80%, the user reviews and manually corrects fields before saving.
2. Email Campaign Creation & Delivery Workflow
Key Points: Email campaign workflow includes creation, composition, scheduling decision, and delivery. After sending, three parallel tracking paths (opens, clicks, engagement scoring) run concurrently. These metrics are aggregated into a final report showing campaign performance metrics like open rate, click rate, and engagement scores.
3. Contact Management & Organization Workflow
Key Points: Contact access can occur through search, browse, or direct favorites access. Once a contact is displayed, users can apply organization actions in parallel (tagging, assignment, editing). The system then checks for duplicates and offers a merge option if found. Finally, all changes are persisted to the database.
4. CRM Synchronization Workflow
Key Points: CRM sync begins with configuration validation and OAuth authentication. On authentication failure, errors are logged and admin is notified. On success, the system queries modified contacts, loads field mappings, transforms data to CRM format, segments into batches, and pushes to CRM API. Finally, sync status and timestamps are updated in the local database.

Activity Diagram Legend

Symbols:

  • Circles (●): Start and end states
  • Rounded Rectangles: Activities/actions performed by the system or user
  • Diamonds (◇): Decision points with branches (Yes/No or other conditions)
  • Lines with Arrows: Flow of execution between activities
  • Color Coding: Blue (user actions), Purple (AI/processing), Green (database/save), Orange (admin actions), Yellow (decisions), Red (errors)
